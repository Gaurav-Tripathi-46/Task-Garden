# Vibe Task Garden

A small full‑stack task board built as a coding assignment.  
Backend uses FastAPI (Python) and frontend uses React + Vite with Tailwind CSS.

## Features

- Add new tasks with a simple input and button.
- See a list of tasks with:
  - Complete checkbox.
  - Delete button.
- Progress indicator showing:
  - Completed / total tasks.
  - Visual progress bar.
- Unique feature: **Plant growth visual**
  - A plant/tree grows as you complete more tasks.
  - Reaches a full tree when all tasks are done.

## Tech Stack

- **Frontend**
  - React + Vite
  - Tailwind CSS
- **Backend**
  - FastAPI (Python)
  - In‑memory storage (no external database)
